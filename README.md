# Talweeh Academy Landing Page

React + Vite 

## Run Locally

1. Open the app folder:

```bash
cd ~/TalweehAcademy/new_talweeh
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```

## Other Commands

Run lint checks:

```bash
npm run lint
```

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Notes

This project uses remote Talweeh Academy image and font assets, so an internet connection is needed for the page to fully match the design locally.
